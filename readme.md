### 当前服务器运营环境：
- 操作系统： CentOS 7.3 x86_64
- WEB环境： Nginx 1.13.0 + php-fpm
- PHP环境： PHP 7.1
  - Zend OPcache
  - apc
  - apcu
  - pdo_mysql
  - redis
  - openssl
  - ...
- 数据库：
  - Percona Server 5.7
  - Redis 3.2
- Node环境：Nodejs 6.10.3
